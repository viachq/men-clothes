# Client - Клієнтський сайт ресторану

Сайт для замовлення їжі онлайн з повним функціоналом.

## ✨ Функціонал

### 🏠 **Головна сторінка**
- Перегляд меню з backend
- Фільтр по категоріях (Перші страви, Десерти, тощо)
- Додавання страв в кошик
- Responsive дизайн

### 🔐 **Автентифікація**
- Реєстрація нових користувачів
- Вхід для існуючих
- Auto-login після реєстрації
- Logout

### 🛒 **Кошик**
- Перегляд товарів в кошику
- Зміна кількості (+ / -)
- Видалення окремих товарів
- Очистити весь кошик
- Автоматичний підрахунок суми

### 📦 **Оформлення замовлення**
- Введення адреси доставки
- Вибір часу:
  - Якомога швидше (30-40 хв)
  - Запланована доставка на конкретний час
- Оплата картою
- Автоматична очистка кошика після оформлення

### 📋 **Мої замовлення**
- Історія всіх замовлень
- Відстеження статусу в реальному часі
- Деталі кожного замовлення (адреса, час, ціна)
- Можливість залишити відгук на доставлене замовлення

### ⭐ **Відгуки**
- Оцінка від 1 до 5 зірок
- Текстовий коментар
- Тільки для доставлених замовлень
- Один відгук на замовлення

## 🔗 Залежності від Backend

Backend має працювати на `http://localhost:8000`

Використовувані API endpoints:
- `POST /auth/register` - Реєстрація
- `POST /auth/login` - Вхід
- `GET /categories/` - Категорії
- `GET /menu/` - Меню (з фільтром по категоріях)
- `GET /cart/me` - Кошик
- `POST /cart/me/items` - Додати в кошик
- `PUT /cart/me/items/{id}` - Змінити кількість
- `DELETE /cart/me/items/{id}` - Видалити з кошика
- `DELETE /cart/me` - Очистити кошик
- `POST /orders` - Створити замовлення
- `GET /orders` - Мої замовлення
- `POST /orders/{id}/review` - Залишити відгук
