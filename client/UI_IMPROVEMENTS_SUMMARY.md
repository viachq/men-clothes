## 1. 🔔 Глобальна система сповіщень (`utils/notifications.ts`)

**Реалізовано:**
- Створено утиліту для показу toast-сповіщень
- 4 типи: `success`, `error`, `info`, `warning`
- Автоматичне видалення попередніх сповіщень
- Плавні анімації появи/зникнення
- Зручні wrapper-функції: `showSuccess()`, `showError()`, `showInfo()`, `showWarning()`

**Використання:**
```typescript
import { showSuccess, showError } from './utils/notifications';

showSuccess('Товар додано в кошик');
showError('Помилка при додаванні');
```

---

## 2. ⏱️ useDebounce Hook (`hooks/useDebounce.ts`)

**Реалізовано:**
- Створено custom hook для debounce значень
- Затримка за замовчуванням: 300ms
- Ідеально для пошукових полів

**Використання:**
```typescript
import { useDebounce } from './hooks/useDebounce';

const [searchQuery, setSearchQuery] = useState('');
const debouncedQuery = useDebounce(searchQuery, 300);
```
