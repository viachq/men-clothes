багатьох редакторах. Для вставлення в пояснювальну записку експортуйте у
PNG/SVG через [mermaid.live](https://mermaid.live) (вставити код блоку) або
розширення Mermaid для VS Code.

## Як експортувати в зображення

1. Відкрити https://mermaid.live
2. Вставити вміст блоку ```mermaid ... ``` з потрібного файлу
3. Export → PNG (або SVG для векторної якості)
4. Вставити зображення у відповідний розділ пояснювальної записки з підписом
   «Рисунок N.M — Назва діаграми»
